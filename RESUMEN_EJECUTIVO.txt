                         🚀 CÓMO INSTALAR Y USAR
═══════════════════════════════════════════════════════════════════════════════

INSTALACIÓN:
1. Conecta tu dispositivo Android o abre un emulador
2. Navega a: app/build/outputs/apk/debug/
3. Ejecuta: adb install app-debug.apk
4. O copia el APK y abre en el dispositivo

PRIMERA EJECUCIÓN:
1. Abre CUICATL en tu dispositivo
2. Autoriza los permisos solicitados
3. La app cargará automáticamente tus canciones
4. Toca una canción para reproducir

USAR REPRODUCTOR:
1. Usa Play/Pause/Stop para controlar
2. Desliza el SeekBar para navegar
3. Lee el tiempo actual y duración

USAR EDITOR:
1. Toca "Editar Audio" en el reproductor
2. Ingresa tiempos en milisegundos (opcional)
3. Aplica efectos deseados
4. Toca Guardar para exportar
