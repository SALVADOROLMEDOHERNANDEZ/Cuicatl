🎯 ¿QUÉ ES CUICATL?
===================

CUICATL es una aplicación Android profesional que incluye:

✅ REPRODUCTOR DE MÚSICA
   - Reproducción de MP3, WAV, OGG, etc.
   - Controles: Play, Pause, Stop
   - SeekBar para navegar por la canción
   - Información de título y artista

✅ EDITOR DE AUDIO PROFESIONAL
   - Recorte de secciones de audio
   - Efecto de eco
   - Efecto de reverberación
   - Normalización de volumen
   - Guardado de archivos editados

✅ DISEÑO FUTURÍSTICO
   - Colores neón (azul cian y púrpura)
   - Tema oscuro profesional
   - Material Design 3


🚀 INSTALACIÓN RÁPIDA (30 SEGUNDOS)
===================================

OPCIÓN 1: Con ADB (Recomendado)
────────────────────────────────
1. Conecta tu dispositivo Android con USB
2. Abre terminal en esta carpeta (CUICATL)
3. Ejecuta este comando:

   adb install app/build/outputs/apk/debug/app-debug.apk

4. La app se instalará automáticamente
5. Abre CUICATL en tu dispositivo
6. ¡Disfruta! 🎵


OPCIÓN 2: Instalación Manual
─────────────────────────────
1. Copia el archivo: app/build/outputs/apk/debug/app-debug.apk
2. Transfiere a tu dispositivo Android
3. Abre con el gestor de archivos
4. Toca "Instalar"
5. Si pide permisos, toca "Instalar de todas formas"

🎵 CÓMO USAR LA APP
===================

PANTALLA PRINCIPAL
──────────────────
- Muestra lista de tus canciones
- Toca una canción para reproducir
- Desliza para ver más canciones


REPRODUCTOR
───────────
▶ Play    → Inicia reproducción
⏸ Pause   → Pausa la canción
⏹ Stop    → Detiene la canción
SeekBar   → Desliza para navegar
Editar    → Va al editor de audio
Atrás     → Regresa a la lista


EDITOR DE AUDIO
───────────────
Recortar Audio:
  - Ingresa tiempo inicial (ej: 5000 ms = 5 segundos)
  - Ingresa tiempo final (ej: 30000 ms = 30 segundos)
  - Toca "Recortar"

Efectos:
  🔊 Eco          → Agrega efecto de eco
  🎼 Reverberación → Agrega reverb
  📊 Normalizar    → Optimiza volumen

Guardar:
  💾 Guardar → Exporta archivo editado
