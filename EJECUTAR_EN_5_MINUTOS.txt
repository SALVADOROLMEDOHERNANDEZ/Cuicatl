═══════════════════════════════════════════════════════════════════════════════
                        ❌ SI ALGO SALE MAL
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: "No aparece el diálogo de selección"
─────────────────────────────────────────────
SOLUCIÓN:
  1. Verifica que el emulador está corriendo
  2. En Device Manager, ¿Ves el botón ▶ o ⏸?
     - Si ves ⏸ = está corriendo ✓
     - Si ves ▶ = no está corriendo, presiona ▶

PROBLEMA: "BUILD FAILED" o error en rojo
──────────────────────────────────────────
SOLUCIÓN:
  1. Build > Clean Project
  2. File > Sync Now
  3. Build > Rebuild Project
  4. Espera a que termine

PROBLEMA: "App no se abre, pantalla negra"
─────────────────────────────────────────
SOLUCIÓN:
  1. Espera 5 segundos
  2. Mira el emulador
  3. Si dice "This app won't run on this device"
     → El emulador es incompatible
     → Crea uno nuevo (ver sección CREAR EMULADOR)

PROBLEMA: "Pide permisos pero no avanzo"
─────────────────────────────────────────
SOLUCIÓN:
  1. En el emulador, presiona "Permitir"
  2. Si no ves botón:
     → Presiona cualquier lugar del emulador
     → Presiona el botón Back
     → Intenta nuevamente

                        🆘 CREAR EMULADOR (si no tienes uno)
═══════════════════════════════════════════════════════════════════════════════

PASO 1: Abre Device Manager en Android Studio
──────────────────────────────────────────────
(Es el botón a la derecha, como se mostró antes)


PASO 2: Haz clic en "+ Create Device"
──────────────────────────────────────

    ┌──────────────────────────────────┐
    │ Device Manager                   │
    ├──────────────────────────────────┤
    │                                  │
    │ [+ Create Device] ← CLIC AQUÍ    │
    │                                  │
    └──────────────────────────────────┘


PASO 3: Selecciona un dispositivo
─────────────────────────────────

Busca y selecciona "Pixel 4" o "Pixel 6":

    ┌──────────────────────────────────┐
    │ Choose a device definition       │
    ├──────────────────────────────────┤
    │ Category: Phone                  │
    │                                  │
    │ ☑ Pixel 4                        │
    │ ☐ Pixel 5                        │
    │ ☐ Pixel 6                        │
    │                                  │
    │    [Next]                        │
    └──────────────────────────────────┘

Presiona [Next]


PASO 4: Selecciona versión de Android
──────────────────────────────────────

    ┌──────────────────────────────────┐
    │ Select a system image            │
    ├──────────────────────────────────┤
    │                                  │
    │ Release Name     API Level       │
    │ ☑ Android 13     API 33          │
    │ ☐ Android 12     API 32          │
    │ ☐ Android 11     API 30          │
    │                                  │
    │    [Next]                        │
    └──────────────────────────────────┘

Elige Android 13 o superior.
Presiona [Next]


PASO 5: Configuración del emulador
──────────────────────────────────

Cuando veas "AVD Name":

    ┌──────────────────────────────────┐
    │ Android Virtual Device (AVD)    │
    ├──────────────────────────────────┤
    │ AVD Name: Pixel 4 API 33         │
    │ Device: Pixel 4                  │
    │ Start up orientation: Portrait   │
    │                                  │
    │ Startup options:                 │
    │ ☑ Use Host GPU                  │
    │                                  │
    │ Memory and Storage:              │
    │ RAM: 4096 MB                     │
    │ VM Heap: 576 MB                  │
    │                                  │
    │ Advanced Settings                │
    │ Keyboard: Use Host GPU           │
    │                                  │
    │   [Cancel] [Finish]              │
    └──────────────────────────────────┘

IMPORTANTE:
  • RAM debe ser 4096 MB mínimo (8192 mejor)
  • ☑ Use Host GPU debe estar CHECKED

Presiona [Finish]


PASO 6: El emulador se crea
──────────────────────────

Espera a que se cree (puede tomar 1-2 minutos).

Luego vuelve a: PASO 1️⃣ (arriba)
