🚀 GUÍA PARA EJECUTAR CUICATL EN EL EMULADOR
=============================================

Sí, la app se mostrará en el emulador. Te muestro exactamente cómo hacerlo.


OPCIÓN 1: DESDE ANDROID STUDIO (Recomendado)
==============================================

PASO 1: Abre el Emulador
─────────────────────────
1. En Android Studio, abre "Device Manager" (lado derecho)
2. Busca un emulador disponible
3. Haz clic en el botón ▶ para iniciarlo
4. Espera a que Android cargue completamente (3-5 minutos)

PASO 2: Ejecuta la App
──────────────────────
1. En Android Studio, abre tu proyecto CUICATL
2. Asegúrate de estar en la carpeta correcta
3. Ve a: Run > Run 'app'
4. O presiona: Shift + F10

PASO 3: Selecciona el Dispositivo
──────────────────────────────────
1. Aparecerá un diálogo "Select Deployment Target"
2. Selecciona tu emulador
3. Haz clic en "OK"

PASO 4: La App Se Instala y Ejecuta
────────────────────────────────────
1. Android Studio instalará el APK
2. La app se abrirá automáticamente
3. ¡Deberías ver la pantalla de CUICATL!


OPCIÓN 2: DESDE LA TERMINAL (Alternativa)
===========================================

PASO 1: Inicia el Emulador
──────────────────────────
Abre PowerShell en la carpeta del proyecto y ejecuta:

  adb devices

Deberías ver tu emulador listado. Si no aparece, inicia desde Device Manager.

PASO 2: Instala el APK
──────────────────────
Ejecuta:

  adb install app/build/outputs/apk/debug/app-debug.apk

Verás: "Success" si se instaló correctamente

PASO 3: Abre la App
───────────────────
Ejecuta:

  adb shell am start -n com.example.cuicatl/.MainActivity

La app se abrirá en el emulador.


¿QUÉ VAS A VER?

Cuando se abra CUICATL en el emulador verás:

1. PANTALLA DE CARGA

PROBLEMAS COMUNES
=================

PROBLEMA: El emulador no inicia
SOLUCIÓN:
1. Abre Device Manager en Android Studio
2. Asegúrate de que la RAM está configurada correctamente
3. Intenta crear un nuevo AVD (dispositivo virtual)
4. Reinicia Android Studio

PROBLEMA: "App no instalada"
SOLUCIÓN:
1. Limpia el proyecto: ./gradlew.bat clean
2. Compila nuevamente: ./gradlew.bat build
3. Intenta instalar el APK nuevamente

PROBLEMA: La app se abre pero no muestra nada
SOLUCIÓN:
1. Espera 5 segundos a que cargue
2. Toca el botón "Permitir" si pide permisos
3. Reinicia la app

PROBLEMA: No veo canciones en la lista
SOLUCIÓN:
1. Es normal, el emulador no tiene música por defecto
2. Puedes agregar canciones manualmente o simplemente probar los controles
3. La app sigue funcionando sin canciones
